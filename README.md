# CSV-AI

CSV-AI is an application built with LangChain, OpenAI, and Streamlit for exploring, summarizing, and analyzing CSV files through natural language and structured workflows.

## Features

- **Interact** — Navigate and query your CSV data via chat.
- **Summarize** — Generate descriptive summaries of CSV contents.
- **Analyze** — Filter, sort, and visualize data with built-in analysis tools.

## Installation

1. Clone the repository and go to the project directory:

   ```bash
   git clone https://github.com/Safiullah-Rahu/CSV-AI.git
   cd CSV-AI
   ```

2. Install dependencies from `requirements.txt`:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

Start the app with:

```bash
streamlit run app.py
```

The app opens in your default browser. Upload a CSV and use the sidebar to switch between Chat, Summarize, and Analyze.
